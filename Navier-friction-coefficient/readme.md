# Usage Steps
Enter the directory LAMMPS and type the following command in the terminal
mpirun -np 8 lmp_mpi -in inpt.lmp -var irun 1
By varying the value for irun you can intialize simulations starting from different intial condition. 
Once the LAMMPS simulations are over go to the POST-PROCESSING directory and enter the folder bot or top depending upon your choice.
